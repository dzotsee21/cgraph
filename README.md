CGRAPH is a simple CLI tool for viewing contributions made on GitHub

commands:<br/>
`cgraph check <github username>` - check any GitHub user's contributions
<br/>
`cgraph checkme` - quickly check your contributions
<br/>
`cgraph change` - change saved username for `checkme` command
<br/>
